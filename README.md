# 薪酬合规审阅工具

一个帮助薪酬合规团队在线审阅 Excel 表格、协作分工、导出带标记结果的轻量 Web 工具。

## 快速启动

```bash
pip install -r requirements.txt
python app.py
```

浏览器打开 `http://localhost:5000`，内网同事可通过你的 IP 访问（需放行防火墙 5000 端口）。

## 功能

- 上传 Excel 自动解析 l1/l2 模块，生成审阅字段
- 三对内容↔依据对照审阅，Tab 切换
- 富文本标记：加粗、修改（红）、新增（绿）、无源（蓝）、删除（线）
- 链接自动识别，点击右半屏打开核对，可标记状态（有效/打不开/内容不符/已过时）
- 按 l1 模块分配校验人，权限隔离
- 一键导出带颜色格式和状态背景色的 Excel
