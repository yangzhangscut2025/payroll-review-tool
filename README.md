# 薪酬合规审阅工具

团队协作的 Web 应用，用于对各国薪酬合规规则进行逐条审阅、标注和导出。

## 访问地址

**生产环境**：腾讯云服务器，8 人同时使用

## 核心功能

- **三格式 Excel 支持** — v1（13列3对）/ v2（7列2对）/ v3（10列2对），上传自动识别
- **审阅工作台** — 三栏布局（左树/Prompt面板可折叠），Tab 切换对照对，Quill 富文本编辑
- **富文本标注** — 加粗、修改（红）、新增（绿）、无源（蓝）、删除线
- **链接管理** — 自动提取链接，状态标注（有效/失效/待验证），纠正链接
- **模块分配** — 按 l1 模块分配校验人，权限隔离
- **AI Prompt 面板** — 一键生成核对 Prompt，**推荐复制到豆包使用**
- **多 Sheet 导出** — 中英文 Sheet 同步导出，英文自动翻译，保留富文本格式
- **导出预览 + 进度条** — 异步导出，实时进度，不阻塞操作
- **AI 用量统计** — 按人/按项目的 Token 消耗和费用估算
- **项目信息编辑** — 名称和国家原地编辑，软删除

## 用户

全部为负责人，用户列表由 `文件说明.xlsx` 的「用户与权限」sheet 定义。
