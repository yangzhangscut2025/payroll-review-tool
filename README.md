# 薪酬合规审阅工具

团队协作的 Web 应用，用于对各国薪酬合规规则进行逐条审阅、标注和导出。

## 快速启动

```bash
pip install -r requirements.txt
python app.py
# → http://localhost:5000
```

内网同事可通过你的 IP 访问（需放行防火墙 5000 端口）。

## 核心功能

- **双格式 Excel 支持** — v1（13列3对照对）和 v2（7列2对照对），上传自动识别
- **审阅工作台** — 三栏布局（左树/Prompt面板可折叠），Tab 切换对照对，Quill 富文本编辑
- **富文本标注** — 加粗、修改（红）、新增（绿）、无源（蓝）、删除线
- **链接管理** — 自动提取链接，状态标注（有效/失效/待验证），纠正链接
- **模块分配** — 按 l1 模块分配校验人，权限隔离
- **AI Prompt 面板** — 一键生成核对 Prompt，**推荐复制到豆包使用**（比 API 直接发送效果好）
- **Excel 导出** — 保留富文本颜色格式和状态背景色
- **AI 用量统计** — 按人/按项目的 Token 消耗和费用估算
- **项目信息编辑** — 项目名称和国家支持原地编辑
