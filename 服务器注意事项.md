# 用电脑做服务器运行审阅工具的注意事项

## 启动

```bash
python app.py
```

同事通过 cpolar 分配的公网地址访问（无需关心本机 IP）。

## 务必做到

| 事项 | 原因 |
|------|------|
| **不要关电脑、不要休眠** | 休眠 = 服务断，所有人都连不上 |
| **cpolar 保持运行** | 退出 cpolar = 外网通道断了 |
| **定期备份 data.db** | 所有数据在一个文件里，每天下班前复制一份 |

## cpolar 注意

- 免费版有带宽限制和域名有效期，确认是否够用
- cpolar 断线会自动重连，但重启电脑后需要重新启动 cpolar
- 可以考虑把 cpolar 和 python app.py 都设为开机自启

## 日常维护

| 操作 | 方法 |
|------|------|
| 改代码需要重启 | 任务管理器杀 python.exe → 重新 python app.py |
| data.db 变大 | 删除旧项目数据 → VACUUM |
| 换端口 | 改 app.py 最后一行 port= 数字，cpolar 配置同步更新 |
