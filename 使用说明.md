# 薪酬合规审阅工具 — 使用说明

## 启动

```bash
cd 项目目录
python app.py
```

浏览器访问 `http://localhost:5000`（内网：`http://你的IP:5000`）

## 基本流程

1. 输入英文名或中文名登录
2. 新建项目 → 填写文件名称和国家
3. 上传 Excel（.xlsx/.xlsm ≤16MB）→ 自动解析 l1/l2 模块
4. 在项目详情页为每个 l1 模块分配校验人
5. 进入审阅工作台 → Tab 切换三对 → 对照审阅
6. 导出审阅版 Excel

## 审阅操作

- **Tab 切换**：顶部 [第1对] [第2对] [第3对] 切换三组对照对
- **状态标记**：面板标题栏右侧点击状态按钮，或快捷键 1/2/3/4
- **编辑文本**：点击右下角 ✎ 按钮 → 面板变为编辑器 → 保存/取消
- **文本标记**：选中文字 + 工具栏按钮
  - B = 加粗 | R = 修改（红色）| G = 增加（绿色）| N = 无源（蓝色）| D = 删除（划线）
- **查看切换**：面板底部「查看原文/查看修改」切换原始和编辑后内容
- **链接操作**：参考依据中链接点击在右半屏打开，可标记状态，可复制
- **侧边栏**：顶栏 « 按钮折叠/展开左侧模块树
- **导航**：←/→ 方向键翻页，树菜单点击跳转

## 权限

- 项目负责人：所有模块可编辑，可分配、上传、删除
- 校验人：仅编辑分配给自己模块
- 登录后自动跳转到第一个分配模块

## 日志

所有请求和错误记录在 `app.log` 文件中。查看：

```bash
notepad app.log    # 打开日志文件
```

每行格式：`时间 [级别] 方法 路径`。日志 10MB 自动轮转，保留最近 3 个备份。
